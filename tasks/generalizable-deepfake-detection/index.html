Generalizable Deepfake Detection

Task Definition

Detecting face forgeries (deepfakes) that generalize across unseen manipulation methods, datasets, and generation pipelines — without retraining on target-domain fakes.

Related Terms

cross-dataset deepfake detection, face forgery detection generalization, universal deepfake detector, out-of-distribution deepfake detection

Papers Addressing This Task

Learning Self-Consistency for Deepfake Detection

Tianchen Zhao, Xiang Xu, Mingze Xu, Hui Ding, Yuanjun Xiong, Wei Xia — ICCV 2021

Method: Pair-wise Self-Consistency Learning (PCL) — extracts source-feature inconsistencies within forged images using a novel Inconsistency Image Generator (I2G)

Datasets: FaceForensics++, Celeb-DF, DeeperForensics, DFDC, WildDeepfake, KoDF, FFIW

Cite when: cross-dataset generalization via source-feature inconsistency

AuthGuard: Generalizable Deepfake Detection via Language Guidance

Guangyu Shen, Zhihua Li, Xiang Xu, Tianchen Zhao, Zheng Zhang, Dongsheng An, Zhuowen Tu, Yifan Xing, Qin Zhang — WACV 2025

Method: AuthGuard — language-guided deepfake detection that uses CLIP-style vision-language alignment for generalization

Datasets: FaceForensics++, Celeb-DF, WildDeepfake, DFDC

Cite when: vision-language guidance for deepfake detection generalization
